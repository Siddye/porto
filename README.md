# 🌸 Raffi Cassidy Chandra's Portfolio

Selamat datang di repositori portofolio pribadi saya! Proyek ini dibangun menggunakan **Vue 3** dan **Vite**, menampilkan profil, *tech stack*, dokumentasi perjalanan ngoding, serta sertifikat kelulusan yang telah saya raih. Desain antarmuka portofolio ini dibuat dengan sentuhan elegan beraksen Jepang (*Japanese Accent*).

## 🚀 Fitur Utama

- **Hero Section**: Memperkenalkan profil saya sebagai Software Engineer & Game Developer, lengkap dengan tautan ke GitHub.
- **Tech Stack**: Menampilkan ikon teknologi yang biasa saya gunakan (HTML5, CSS3, JavaScript, Python, Java, PHP).
- **Galeri Dokumentasi**: Slider gambar interaktif yang dilengkapi fitur *Lightbox* untuk melihat lebih jelas dokumentasi hasil ngoding sehari-hari.
- **Sertifikat & Pencapaian**: Daftar sertifikat pemrograman yang dapat diakses dan diunduh langsung.
- **Responsif & Estetik**: Dilengkapi animasi *orb* di latar belakang, efek *hover* yang mulus, dan antarmuka yang ramah pengguna.

## 🛠️ Teknologi yang Digunakan

- **[Vue 3](https://vuejs.org/)** - Menggunakan *Composition API* dengan sintaks `<script setup>`.
- **[Vite](https://vitejs.dev/)** - *Build tool* yang sangat cepat dan modern untuk pengembangan web.
- **CSS3** - Animasi *custom*, transisi, dan desain responsif murni.

## 📦 Panduan Instalasi (Local Setup)

Untuk menjalankan proyek ini di perangkat Anda sendiri, silakan ikuti langkah-langkah berikut:

1. **Clone Repositori**
   ```bash
   git clone https://github.com/Siddye/portfolio.git
   cd portfolio
   ```
   *(Catatan: Sesuaikan URL clone dengan link repositori Anda yang sebenarnya jika berbeda).*

2. **Install Dependensi**
   Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/), lalu jalankan perintah ini:
   ```bash
   npm install
   ```

3. **Jalankan Server Mode Development**
   ```bash
   npm run dev
   ```
   Buka URL yang tertera di terminal (biasanya `http://localhost:5173/`) di browser Anda.

4. **Build untuk Produksi**
   Jika ingin melakukan *deployment*:
   ```bash
   npm run build
   ```

5. **Preview Hasil Build**
   ```bash
   npm run preview
   ```
